# References

## FAQ
- SQLiteは使える? → 使用禁止。開発環境を含めPostgreSQLのみ。
- 画像の保存先は? → 開発は`public/uploads`、本番はVercel BlobやS3等を推奨。
- ユーザー管理は? → ADMIN権限の管理者コンソールから可能。
